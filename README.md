# DataMart - Premium Marketplace Script

Welcome to DataMart! This is a complete, production-ready multivendor marketplace script built with Next.js App Router, Tailwind CSS, Shadcn UI, Neon DB, Drizzle ORM, and NextAuth.js.

## What is DataMart?
DataMart is a turnkey solution for entrepreneurs to launch their own digital asset marketplace. Vendors can register, upload digital products (stored securely on Cloudflare R2), and get paid via Stripe and Binance Pay.
